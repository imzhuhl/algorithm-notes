
**思路**：

此题可以维护一个递减的单调队列，队首元素就是最大值。

问题在于如何维护一个递减的队列，使用到双向队列，针对上述例子：

```
[1 3 -1] -3 5 3 6 7     [3, -1]
1 [3 -1 -3] 5 3 6 7	    [3, -1, -3]
1 3 [-1 -3 5] 3 6 7	    [5]
...
```

首先，队列必须是递减的，例如一开始 1 入队列，但是下一个入队的 3 大于 1，因此 1 必须出队列。因为 1 首先入队列，并且比 3 要小，可以认为在之后的过程中，只要 3 一直在滑动窗口内，最大值就不可能是 1，因此可以将 1 出队列。
